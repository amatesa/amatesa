# 00 — READ FIRST

## Welcome

This repository contains the complete specification for Alejandro Matesa's personal portfolio.

This is not a template.

It is not a traditional portfolio.

It is a content-driven interactive product designed to communicate how Alejandro thinks, solves problems and builds digital experiences.

The documentation is intentionally divided into layers.

Do not begin implementation before understanding the product vision.

---

# Repository Philosophy

The portfolio follows four fundamental principles:

* Understand before building.
* Build systems, not isolated pages.
* Keep content independent from presentation.
* Prioritize clarity over complexity.

Every implementation decision should reinforce these principles.

---

Do not prioritize visual effects over communication.
