# 07 — Portfolio Copy Master

## Purpose

This document contains every piece of text used throughout the portfolio.

It is the single source of truth for all interface copy.

Whenever possible, components should reference this document rather than invent new wording.

The goal is consistency.

---

# Writing Principles

Every sentence should satisfy at least one of these purposes:

Explain.

Guide.

Invite.

Demonstrate.

Never write text simply to fill space.

---

# Voice

Thoughtful.

Curious.

Honest.

Clear.

Confident without arrogance.

Professional without sounding corporate.

Human without becoming informal.
